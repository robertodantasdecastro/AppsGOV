1.2 - Apache<br>
1.3 - Mysql<br>
1.4 - Sistemas Operacionais<br>
1.5 - Browsers<br>
1.6 - Bibliotecas</p><p>2 - Instruções de Instalação</p><p>2.1 - Estrutura<br>
2.2 - Instalações Necessárias<br>
2.3 - Configurações Connect<br><br><br>
=====================================================<br><br><br>
1.1 - PHP<br><br>
    Versão PHP: 5.1.6<br>
    Configurações do arquivo php.ini<br>
        short_open_tag:    On<br>
        register_globals: On<br>
        upload_max_filesize: 8M<br>
        Default timezone: America/Sao_Paulo<br>
        magic_quotes_gpc: ON<br>
        output_buffering: 4096<br>
        error_reporting  =  E_ALL &amp; ~E_NOTICE<br>
        display_errors = On</p><p>1.2 - Apache<br><br>
    Versão Apache: 2.2.3<br>
    Apache API Version: 20051115<br>
    HTTP_ACCEPT_CHARSET: ISO-8859-1</p><p>1.3 - Mysql<br><br>
    Mysql versão 5.0.77</p><p>1.4 - Sistemas Operacionais<br><br>
    Linux<br>
    Microsoft Windows(R)</p><p>1.5 - Browsers<br><br>
    Recomendações:<br>
        Firefox v 3.5 ou superior<br>
        Chrome v 5.0 ou superior<br>
        Internet Explorer v 7 ou superior</p><p>1.6 - Bibliotecas<br><br>
    Lib GD for PHP5 bundled (2.0.28 compatible)</p><p>==================================================================</p><p>2 - Instruções de Instalação<br><br><br>
2.1 - Estrutura de pastas<br><br>
    Linux:<br>
        /enota/site    - interface website para prestador e tomador de serviços<br>
        /enota/sep    - interface para gestão da informação pelo fiscal tributário<br><br>
    Windows:<br>
        C:\enota\site    - interface website para prestador e tomador de serviços<br>
        C:\enota\sep    - interface para gestão da informação pelo fiscal tributário</p><p>2.2 - Instalações Necessárias<br><br>
    a) Instale o Apache (apache.org), PHP (php.net) e MySQL (mysql.com) de acordo com as versões exigidas acima.<br>
    b) Importar arquivo *.sql da base modelo, recomendamos que utilize o PhpMyAdmin (www.phpmyadmin.net)<br>
    c) Após a instalação do MySQL e importação da base modelo, criar usuário e senha para acesso a base criada.</p><p>2.3 - Configurações config.php<br><br>
    Localização do arquivo: eiss/include/config.php<br>
    Necessária adaptação das seguintes variáveis:<br>
        - $Host : é o endereço da máquina servidor;<br>
        - $Usuario : usuário de conexão ao banco MySQL;<br>
        - $Senha : senha do usuário de conexão ao banco MySQL;<br>
        - $Base : base selecionado na conexão.</p><p>www.portalpublico.com.br}</p>
